# 🔐 Animated Login & Registration Page

A modern **animated authentication interface** built using **HTML, CSS, and JavaScript**.
This project showcases a smooth **UI transition between Login and Sign-Up forms** using CSS animations and JavaScript for interactivity.

The interface focuses on **clean design, responsive layout, and engaging user experience**, making it a great practice project for front-end development.

---

## ✨ Features

* 🔐 **Login and Registration forms**
* 🔄 Smooth **animated transition** between Sign In and Sign Up
* 🎨 Modern UI with styled form components
* 🌐 **Social media icon placeholders**
* 📱 Responsive layout for different screen sizes
* ⚡ Lightweight and built with **vanilla JavaScript**

## ⚙️ How It Works

1. The page contains **two forms**:

   * Sign In
   * Sign Up

2. A **panel container** holds animated sections.

3. When a user clicks:

   * **Sign Up** → the interface slides to the registration form
   * **Sign In** → the interface slides back to the login form

4. CSS transitions handle the **smooth sliding animation**.
